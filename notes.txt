[+] Set up virtual env
=========================================================================================
    - Navigate to project folder/repo
    - Create env
    -> python -m venv env
    - Activate virtual env
    -> source env/bin/activate
    - Deactivate virtual env
    -> deactivate
-----------------------------------------------------------------------------------------
[+] Install flask
=========================================================================================
    - pip install flask
-----------------------------------------------------------------------------------------
[+] Run app with env var set to flask app
=========================================================================================
    - Set main flask file/app as flask app for env var
    -> export FLASK_APP=file_name.py
    ->> export FLASK_APP=hall_pass.py
    - Run app
    -> flask run
    - Set to run in debug mode
    -> export FLASK_DEBUG=1
-----------------------------------------------------------------------------------------
[+] Run app by filename without env vars
=========================================================================================
    - Navigate to bottom of main app file and add
    if __name__ == '__main__':
        app.run(debug=True)
    -> python file_name.py
    ->> python hall_monitor.py
-----------------------------------------------------------------------------------------
[+] wtf-forms
=========================================================================================
    - pip install wtf-forms
